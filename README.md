# Kids School Website

Welcome to the **Kids School Website**! This project is a simple, visually engaging website designed for kids' educational purposes.

## Live Demo

Check out the live website here: [Kids School Website](https://riyad899.github.io/Kids-School/)

## About the Project

This project is built using:

- **HTML**: For structuring the content.
- **CSS**: For styling and making the website visually appealing.

### Features

- User-friendly design tailored for kids.
- Eye-catching and colorful layout.
- Interactive and educational elements.

## Development Details

- **No frameworks used**: This project is created entirely with vanilla HTML and CSS, ensuring simplicity and full control over the codebase.
- Fully responsive design to ensure it works across various devices.

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/riyad899/Kids-School.git
   ```

2. Open the `index.html` file in your favorite browser to view the website.
